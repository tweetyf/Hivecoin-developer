setnetworkactive
================

``setnetworkactive state``

Disable/enable all p2p network activity.

Argument #1 - state
~~~~~~~~~~~~~~~~~~~

**Type:** boolean, required

true to enable networking, false to disable

Result
~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - true|false
     - boolean
     - The value that was passed in
